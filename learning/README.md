# Neural Network Learning Guide

This folder organizes the main neural network families we will study in PyTorch.

## Folders

- `mlp`: Feedforward multilayer perceptrons for tabular data and basic classification
- `cnn`: Convolutional neural networks for images and spatial data
- `rnn`: Basic recurrent neural networks for sequence modeling
- `lstm`: Long short-term memory networks for longer sequence dependencies
- `transformers`: Attention-based models for modern sequence tasks

## Suggested order

1. `mlp`
2. `cnn`
3. `rnn`
4. `lstm`
5. `transformers`
